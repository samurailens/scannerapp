About

This App provides basic functionality to scan any kind of barcode with inbuilt camera. The data is erased once the app is closed. To use the scanned information with server, create an API endpoint on server and use it in the app in RestApi provider. See the sample provided in rest-api.ts file.

Supported Barcode Formats

'QR_CODE' | 'DATA_MATRIX' | 'UPC_E' | 'UPC_A' | 'EAN_8' | 'EAN_13' | 'CODE_128' | 'CODE_39' | 'CODE_93' | 'CODABAR' | 'ITF' | 'RSS14' | 'RSS_EXPANDED' | 'PDF417' | 'AZTEC' | 'MSI'
